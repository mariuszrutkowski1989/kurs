DOM - Document Object Model

Keyborad events:

  • keydown - ANY key is pressed (including CTRL, ALT, TAB, SHIFT, ESC etc.) - this event always fires FIRST. (Fired continously)
  • keypress - ANY key (except CTRL, ALT, TAB, SHIFT, ESC etc.) is in pressed position. (Fired continously)
  • keyup - ANY key is released
